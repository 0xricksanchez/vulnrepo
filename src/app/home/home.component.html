# Create vulnerability reports like a PRO!!!

VULNREPO is a free project designed to speed up the creation of IT Security: vulnerability reports.
Focus on research, spend time on research NOT on the report form!

Feature list

security Security

Project use browser for encrypt/decrypt (AES256) & store in local. Full confidentiality of data, nothing is sent out.

share Share report

You can share your report using AES encryption by default.

cloud_download TXT & HTML & PDF

You can download report in TXT, HTML formats!. If you need PDF just 'print as PDF' html report.

 

info Use custom issues templates!

The use of templates greatly speeds up the work for pentester or security auditor.

info Import issues from security scanners

After importing, easily manage and edit vulnerabilities.

track_changes Changelog

All important changes in report will be logged in to changelog and update to the next version of the report.
