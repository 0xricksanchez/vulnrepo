FAQ

Browser support

Project is working on latest chrome/firefox browser.

 How reports are stored?

All data are stored in browser (on local machine, nothing is send outside) using indexeddb and SessionStorage, reports by default are encrypted in AES.

 How to restore my encrypted report if I forgot my security key?

Use bruteforce method or better remember the security key, otherwise there is no other option to decrypt data.

 Can I change report security key?

If you able to decrypt report where security key you want to change, yes!.

 How to create PDF report?

Download HTML report, open and use 'print to pdf' feature in your browser (ctrl+p): Chrome/FireFox.
